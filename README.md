# 🛡️ Jeremy Wood | Technical Sentinel Portfolio

This repository contains the source code for [jeremywood.digital](https://jeremywood.digital), the professional portfolio of Jeremy Wood—a GTM Systems Architect and Technical Solutions Engineer.

## 🎯 The Mission
The "Technical Sentinel" aesthetic represents a commitment to technical forensics, data integrity, and architectural precision in Revenue Operations and GTM systems.

## 🏗️ The Stack
- **Architecture:** Pure HTML/JS with a focus on speed and SEO.
- **Styling:** Custom Vanilla CSS with "Sentinel Dark" aesthetics.
- **Tracking:** High-fidelity GA4 and Server-Side GTM integration (G-DNJTR4388W).
- **Automation:** Integrated with the **Vault Distiller MCP** for automated technical context surfacing.

## 📁 Repository Structure
- `index.html`: The core portfolio landing page.
- `styles.css`: The "Sentinel" design system.
- `vault-distiller-mcp/`: Source code for the MCP server used to automate technical distillation.
- `applications/`: Specialized application materials and case studies.

## 🚀 Key Projects
- **The Forensic Vault Distiller (MCP):** Bridging the gap between technical vaults and the Buffer API.
- **Ancestry Data Forensics:** A high-fidelity data aggregation project leveraging Markdown-based relational modeling.
- **Handheld Storage Optimizer:** Data-driven utility for optimizing storage allocation in embedded systems.
- **Nomad Flight Intelligence:** Python-based deal scanning for family travel.
