# Jeremy Wood | Technical AdOps Portfolio

This is the pure HTML/CSS/JS codebase for the lightweight, high-performance portfolio website explicitly designed for Technical AdOps and Implementation role targeting.

## Deployment Instructions (Vercel)
This codebase is completely ready for a 1-click launch.

1. Create a free account at [Vercel](https://vercel.com/)
2. Drag and drop this entire `portfolio` folder directly into the Vercel dashboard to deploy it instantly.
3. In Vercel, go to the "Domains" tab and link your Porkbun domain (`jeremywood.dev`).

### Editing the Content
Since there is no backend CMS, all content changes are made directly in `index.html`. 
- **Resume**: The `Jeremy_Wood_AdOps_Resume.pdf` file is included in the root folder so the "Resume" download button works natively once deployed. If you update the resume, overwrite this file before deploying.
- **Theme**: Colors and animations can be edited inside `css/styles.css`.
