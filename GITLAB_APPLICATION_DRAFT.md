### 4. Tell us about a time you solved a complex technical problem for a customer.

"My 'customers' are high-velocity publishers who rely on complex ad-tags. I once troubleshot a case where a publisher’s revenue dropped to zero after a 'minor' site update. By using Chrome DevTools and the 'Network' tab, I identified that a new lazy-loading script was accidentally suppressing the 'Request for Ad' (RFA) pixel by firing it in a hidden container. 

Instead of just reporting a bug, I cloned the page, modified the script locally to use a standard IntersectionObserver, and proved that the revenue would return with this change. I value 'Results' over 'Activity'—my goal is always to get the user back to full functionality as fast as possible while documenting the fix so it never happens again."
